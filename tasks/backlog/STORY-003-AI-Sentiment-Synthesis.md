# STORY-003: AI Sentiment & Synthesis Engine

**Description:** As a user, I want the system to analyze daily news and synthesize a cohesive economic thesis.
**Requirements:**
- Ingest financial news headlines.
- Pass headlines and macro data to LLM.
- Output thesis and 3-4 recommended asset classes.
**Acceptance Criteria:**
- Unit tests verify prompt construction and LLM API response handling.
- Functional tests verify the generated report format.
