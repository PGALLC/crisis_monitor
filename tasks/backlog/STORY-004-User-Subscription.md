# STORY-004: User Subscription & Preference Center

**Description:** As a user, I want to subscribe, verify my contact info, and set my report frequency.
**Requirements:**
- Registration with Email/Phone verification loop.
- Preference selection (Daily/Weekly, Urgent Alerts, Events).
- Secure, tokenized unsubscribe flow.
**Acceptance Criteria:**
- Unit tests cover token generation and validation.
- Functional tests simulate the verification loop and preference updates.
